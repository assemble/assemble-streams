## Usage

This plugin can be used with [assemble][], [verb][] or any other application based on [templates][].

```js

var streams = require('{%= name %}');

// create your application
var app = assemble();
// register the plugin
app.use(stream); 

// use the plugin
app.toStream('pages')
  .pipe(app.dest('site/'))
```
